# Akash Singh - Portfolio

This is a personal portfolio website built with HTML, CSS, and JavaScript.

## Structure

- `index.html`: Main content of the website.
- `style.css`: Styles and theming (Dark mode).
- `script.js`: Interactivity (Mobile menu, smooth scrolling, animations).

## How to Run

Simply open `index.html` in any modern web browser.

## Customization

- Update the links in the `social-links` section in `index.html`.
- Replace the `#` in `href` attributes with your actual profile URLs.
